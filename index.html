Un moteur de recherche vient en supplément de BDD existantes (relationnelle, NoSQL, fichier...)

Indexation

Trois procédés majeurs d'indexation existent :

  • Différé complète : un bach d'indexation ré-indexe toutes les données à interval régulier
  • Différé incrémental : un bach d'indexation indexe les nouvelles données et les données modifiés à interval régulier
  • Direct : lors de la mise à jours des données dans les applications, mettre à jours les données dans Solr
